Automated Produce Peeler

APP is a tool to sort your produce by type and condition (good/bad) and peel it!

SUCCESS

How We Implemented This Product

Using image processing, python GUIs, PCB design, and power electronics!

Image Processing

Using our trained dataset, we predict the produce type as well as whether the produce is defective or not.

Graphical User Interface (GUI)

Using python to implement the graphical user interface which allows individuals to select the produce type they want peeled.

PCB Design

PCB designed to support and connect electrical components including the DC and stepper motor drivers, and LCD display.

Motors

DC motor used to control the turntable and stepper motor to control the vertical position of knife.
